# Shuffle
```
【製作期間】	5時間
【使用言語】	C言語(BetterC)
【ジャンル】	トランプのシャッフル(重複のない乱数を生成するツール)
【概要】	トランプを題材に重複がないように乱数を生成しました。
【操作方法】	起動するだけです。
【書いた範囲】	Cコードのすべて
【制作動機】	ブラックジャックを作る前段階として作りました。
【頑張った所】	A(1),J(11),Q(12),K(13)を解釈させるところが少し大変でした。
【面白いところ】何番目にあるか、ということを大切にし、また重複の無いように乱数を順番に入れていくのではなく、
		逆に乱数の場所に現在のカウンタの数値を入れていく、という特殊な処理をしております。
		それによってより”トランプ”感が出たと思います。
```
